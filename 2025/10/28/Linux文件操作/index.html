vim命令

vim是Linux下功能强大的文本编辑器，支持多种编辑、查找、替换和多窗口等操作。基本用法和常见操作如下：

启动和退出vim

退出vim的方法：

  • :q # 退出（若有未保存内容则会提示）
  • :q! # 强制退出不保存
  • :wq # 保存并退出
  • :x # 保存并退出，等同于:wq
  • ZZ # 保存并退出（在命令模式下输入）

只保存文件：

  • :w # 保存文件

vim三种模式

  • 普通模式（Normal）：默认进入，进行复制、粘贴、删除等操作。按Esc键进入普通模式。
  • 插入模式（Insert）：输入文本。按i、I、a、A、o、O进入插入模式。
  • 命令模式（Command）：用于执行保存、退出、查找、替换等命令。按:进入命令模式。

常用操作

插入模式切换

  • i # 在光标前插入
  • I # 在行首插入
  • a # 在光标后插入
  • A # 在行尾插入
  • o # 在当前行下方新建一行并插入
  • O # 在当前行上方新建一行并插入

光标移动

  • h # 左移
  • j # 下移
  • k # 上移
  • l # 右移
  • w # 移动到下一个单词首
  • b # 移动到上一个单词首
  • 0 # 移动到行首
  • $ # 移动到行尾
  • gg # 移动到文件首
  • G # 移动到文件尾
  • numG # 移动到第num行

复制、粘贴与删除

  • yy # 复制当前行
  • nyy # 复制接下来的n行
  • p # 在光标后粘贴
  • P # 在光标前粘贴
  • dd # 删除当前行
  • ndd # 删除接下来的n行
  • x # 删除光标处字符
  • d0 # 删除至行首
  • d$ # 删除至行尾
  • :%d #删除文件中的所有行。
  • ggdG #从文件头移动到末尾并删除所有内容。

撤销与重做

  • u # 撤销
  • Ctrl+r # 重做

查找与替换

  • /str # 向下查找字符串str
  • ?str # 向上查找字符串str
  • n # 重复上一次查找（同方向）
  • N # 重复上一次查找（反方向）

替换命令：

  • :s/old/new/ # 替换当前行第一个old为new
  • :s/old/new/g # 替换当前行所有old为new
  • :%s/old/new/g # 替换整个文件所有old为new
  • :%s/old/new/gc # 替换整个文件所有old为new，每次替换前确认

多窗口与多文件

  • :split file2 # 水平分割窗口并打开file2
  • :vsplit file3 # 垂直分割窗口并打开file3
  • Ctrl+w w # 切换窗口
  • :tabnew file4 # 新建标签页并打开file4
  • :tabn # 切换到下一个标签页
  • :tabp # 切换到上一个标签页

行号显示

  • :set number # 显示行号

  • :set nonumber # 关闭行号

其他实用命令

  • :e file5 # 打开file5
  • :r file6 # 将file6内容插入到当前文件
  • :! command # 执行shell命令（如:!ls）

vim操作非常丰富，上述为常用基础命令，熟练掌握可大幅提升文本编辑效率。
